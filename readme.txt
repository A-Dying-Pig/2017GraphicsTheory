1.打开文件夹graph,打开其中的graph_user.exe程序,需要等待1分钟左右初始化程序。
程序初始化完成后会弹出一个交互对话框，输入相应点的数据并点击对应按钮就可以弹出图的页面。
鼠标移动到图中的点上可以显示对应点的信息，不同的图的交互略有差别，具体请见相应图中的介绍。

2.本程序既使用了Qt也使用了d3，在graph文件夹下，一些以.dll为后缀的文件和一些文件夹是Qt程序运行所需要的链接库，
”graph_user.exe”是主程序,”newuserMF.csv”是处理后的”user.csv”，文件夹html中存放着使用d3的网页，
html中的字文件夹js中存放着d3所需要的库和图正常运行的数据(.js的格式)。
